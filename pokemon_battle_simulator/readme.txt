Pokemon Battle Simulator

Description:
This project simulates a turn-based Pokémon battle between two trainers. Each trainer picks Pokémon, which take turns attacking. The first trainer to knock out all opponent Pokémon wins.

Files:
- battle.py → The main driver file that runs the battle.
- pokemon.py → Defines the Pokemon class.
- trainer.py → Defines the Trainer class.
- battle_log.py → Handles logging battle events.
- battle_log.txt → Stores battle history.

How to Run:
1. Make sure all .py files and battle_log.txt are in the same folder.
2. Open your terminal/command prompt.
3. Navigate to the folder where files are located.
4. Run the game with the following command:
   python battle.py
